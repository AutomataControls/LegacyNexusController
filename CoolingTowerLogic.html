Control Function Overview

processCoolingTowerControl(data, uiCommands, stateStorage)

The main control function processes incoming sensor data and generates control commands for cooling tower systems with proper staging and safety interlocks.

Input Parameters

  • data: Sensor data from cooling tower systems
  • uiCommands: User interface commands and setpoints
  • stateStorage: Persistent state storage for timing and cycling

Control Outputs

  • VFD Enable signals (T1-T3)
  • Speed references (AO1-AO3, 0-10V)
  • Bypass valve control (AO4, 0-10V)
  • Heater controls (CH1-CH3)
  • Isolation valve controls (CH4-CH9)
  • System status and alarm conditions
